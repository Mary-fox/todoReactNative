# todoReactNative

запуск веба
npx expo start --web

\*пометка
| `@react-navigation/native` | база навигации |
| `native-stack` | экраны как “стек страниц” |
| `react-native-screens` | ускорение экранов |
| `gesture-handler` | свайпы и жесты |
| `reanimated` | анимации переходов |
| `safe-area-context` | безопасные зоны экрана |

запуск в xcode
npx expo prebuild
open ios/\*.xcworkspace
